DRAG AND DROP TO GITHUB PAGES
=============================

Upload ALL of these files to the ROOT of your gh-pages branch (or main):

  index.html
  styles.css
  about.html
  about-the-app.html
  stories.html
  blog.html
  blog-combining-points.html
  blog-romance-dead.html
  blog-behavioral-cues.html
  blog-long-distance.html

1. GitHub → your repo → create/switch to branch "gh-pages"
2. Drag and drop the 10 files above into the repo root
3. Settings → Pages → Source: branch gh-pages, / (root)
4. Site: https://YOUR_USERNAME.github.io/YOUR_REPO/

Custom domain: Settings → Pages → Custom domain → add domain, then CNAME at DNS to YOUR_USERNAME.github.io

To embed the app: when deployed, edit about-the-app.html — set the iframe src to your app URL and remove the app-embed-placeholder div.
